# Changelog

## v0.2.0

- 重构 `main.py`，统一为可编译、可维护的远端 `memes API` 调用逻辑
- 修复命令名、帮助文本、错误提示和详情文本中的乱码问题
- 新增 `meme_ping`、`meme帮助`、`meme列表`、`meme搜索`、`meme更新`、`随机meme` 指令说明
- 优化被动关键词触发逻辑：
  - 关键词必须位于消息开头
  - 需要文字输入的模板必须使用 `关键词 + 空格 + 文本`
  - 不需要文字的模板不再因为普通聊天尾随文本而误触发
  - 按模板最小图片数和最小文本数做预检查
- 为需要图片的模板增加触发者头像保底机制
- 增加引用消息图片、直接图片、`@` 用户头像的输入收集逻辑
- 增加模板黑白名单过滤功能
- 增加 HTTP 超时与重试配置：
  - `http_timeout_seconds`
  - `http_retry_times`
- 完善模板缓存逻辑：
  - 优先读取本地 `infos.json` / `key_map.json`
  - 远端更新失败时保留本地缓存
- 重写 `README.md`，补充安装、配置、指令、触发规则与常见问题说明
- 修复 `metadata.yaml` 与 `_conf_schema.json` 的乱码和格式问题

## v0.1.0

- 初始化 AstrBot 插件基础结构
- 接入远端表情包生成 API
